# Crystal Open Weather Map Interface

A Crystal interface for the Open Weather Map API.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  open_weather_map:
    github: [HCLarsen]/open_weather_map
```

## Usage

```crystal
require "open_weather_map"
```
